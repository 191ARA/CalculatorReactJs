Документация к Calculator

Calculator представляет собой калькулятор, разработанный с использованием React. Он поддерживает базовые математические операции, обработку нажатий клавиш и специальные функции, такие как возведение в квадрат, куб и обратное число.

Используемые технологии
React: для управления состоянием и рендеринга пользовательского интерфейса.
Tailwind CSS: для стилизации элементов интерфейса.

Логика работы

Состояние
input (useState): хранит текущее выражение, введенное пользователем.

Эффекты
useEffect: отслеживает нажатия клавиш и вызывает соответствующие обработчики при нажатии цифровых и операторных клавиш.

Основные функции

handleKeyPress(event):
Фильтрует нажатые клавиши, разрешая только допустимые (0-9, +, -, *, /, ^, Enter, Backspace).
При Enter вызывает handleClick("=").
При Backspace удаляет последний символ в input.

isValidInput(value):
Проверяет, можно ли добавить символ в input.
Запрещает ввод подряд идущих операторов.

handleClick(value):
Выполняет действия в зависимости от переданного значения:
C: очистка input.
=: вычисление результата выражения с помощью eval() (заменяя ^ на **).
x²: возведение текущего значения в квадрат.
x³: возведение текущего значения в куб.
1/x: вычисление обратного значения.
Любое другое значение добавляется в input, если оно допустимо.

Разметка

Основная обертка div с классами flex flex-col items-center bg-gray-800 p-6 rounded-2xl shadow-lg w-72.
Поле отображения ввода: div с классами mb-4 p-4 bg-black rounded text-right text-2xl h-16 flex items-center justify-end.
Кнопки калькулятора размещены в сетке grid grid-cols-4 gap-2, каждая кнопка оформлена с классами bg-gray-700 p-4 rounded-lg text-white text-xl hover:bg-gray-600.
Кнопка C выделена красным цветом bg-red-600.

Рендеринг
Калькулятор рендерится в корневой элемент с помощью ReactDOM.createRoot(document.getElementById("root")).render(<Calculator />);.
